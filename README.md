# Mental Health Survey Analysis using SQL

## Business Problem
Analyze workplace mental health survey data to understand treatment patterns.

## Tools Used
- SQL
- SQLite
- DB Browser for SQLite

## Analysis Performed
- Removed unrealistic age values
- Calculated total survey responses
- Analyzed treatment distribution
- Examined treatment among tech employees
- Compared remote vs onsite workers

## Key Insights
- Many respondents reported taking mental health treatment.
- Treatment trends were analyzed among tech employees.
- Remote work patterns were compared with treatment behavior.
